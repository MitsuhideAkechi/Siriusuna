# 「ここはどこ」

你可以在这里找到:  
- 我学习的一些国外 CS 课程的笔记
- 我在学校学习的一些课程的笔记
- 我在一些学习过程中产生的副产品 (我觉得写都写了, 不放上来镇场面不就白写了的东西, 比如 [CO 的设计文档](https://siriusuna.top/BUAA-CO).)
- 我的一些日常想法, 博文一类的东西
- 我可能会有的一些日语习作

与其说是博客, 更应该称之为数字花园吧? 里面目前记录的有我学习各种各样知识顺势而为自己做的笔记, 原本都没考虑过公布出来, 不然我一定会做的更好看更用心的...

可惜以前完全没考虑过有 [Quartz](https://github.com/jackyzha0/quartz) 这样方便的框架呢! 对 Obsidian 的支持惊人的好, 几乎是一键部署就能在网页很好地显示并且使用双链了. 我白白为了在 GitHub 上显示图片双链, 特地用几个脚本来整理目录和引用方式, 还给每个文章标题的空格改成了难看的下\_划\_线.

总之我会逐渐检查整理以前的笔记, 修复他们的标题, 增加元信息配置啦, 回顾一下自己一年以来的学习.
